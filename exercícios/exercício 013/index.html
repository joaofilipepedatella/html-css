CSS Estilo Inline

Usando o estilo inline

No estilo inline você precisa adicionar os parâmetros manualmente em cada linha que fizer.
Por Exemplo:
<body style= "background-color: lightblue; font-size: 20px"
Quando houver mais de um parâmetro na tag STYLE, separá-los por ponto-vírgula (;)

Usado apenas para pequenas modificações, não no dia-a-dia em um trabalho completo, pois polui as linhas de código.
