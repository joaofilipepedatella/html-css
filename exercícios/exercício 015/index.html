CSS Estilo Externo

Usando o estilo externo

No estilo externo do CSS você faz as linhas de comando dentro de um arquivo criado com o link:css, clicando com o CTRL apertado no nome do valor em href. Dentro deste novo arquivo, você faz o Style, e apenas vai precisar colocar a linha de link:css no HEAD das suas páginas.
