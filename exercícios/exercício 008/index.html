Usar marca texto

Usar a tah MARK na linha de comando fará que o texto fique com forma de marca texto

Criar texto grande e pequeno

A tag BIG faz com que o texto aumente o tamanho porém esta tag está obsoleta, e isso é feito no CSS agora

a tag SMALL diminui o tamanho do texto e ainda está em uso pelo HTML

Texto deletado

a tag DELL faz com que o texto fique riscado indicando assim que o texto foi excluído, porém ele pode ser lido para compreensão.

Texto inserido

a tag INS vai sublinhar para indicar que o texto foi inserido (com semântica) e é utilizado pelo html

já a tag U, vai sublinhar o texto (sem semântica) e não é mais utilizado, sendo assim, obsoleto.

Texto sobrescrito

Use a tag SUP para escrever o texto na parte superior da frase. Exemplo: o professor pediu para resolver a equação x2+2xy-y2=0

Texto subscrito

use a tag SUB para subscrever o texto, ficar na parte inferior da frase. Exemplo: A fórmula da água é H2O.
