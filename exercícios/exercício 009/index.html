Listas Ordenadas e Não Ordenadas

Listas Ordenadas OL

São listas em que a ordem importa. Utiliza-se a Tag OL para que o comando já enumere a lista de acordo com a ordem colocada no código. A ordem dos itens é feita pela Tag LI. Por Exemplo:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café da manhã
  4. Ir trabalhar
Dentro da Tag OL, caso queira mudar o ícone da lista, o parâmetro TYPE vai oferecer alguns valores para que seja utilizado nas listas.
  1. Almoçar
  2. Voltar ao trabalho
  3. Ir à academia
  4. estudar
  5. Dormir

Listas não ordenadas

Lista em que a ordem não é importante. É usada a Tag UL para a criação da lista, e a tag LI para a enumeração.

    Lista de compras
  • Arroz
  • Feijão
  • Batata
  • Carne

Juntando Listas

Para criar uma lista dentro de um LI, basta criar outro OL dentro da mesma e enumerar com outro LI

    Carnes
  1. Bovinas
  2. Suínas
  3. Frango

Caso precise fazer duas listas em que uma complementa a outra, no segundo OL (e subsequentes) colocar o parametro START e no valor usar o NÚMERO que deseja continuar a ordem.
