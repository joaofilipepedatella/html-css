Outras Formatações

Código-Fonte / Pré-Formatação

A Tag CODE faz com que o texto tenha sua formatação usando o mesmo espaçamento para todo o corpo, tanto em letras, quanto em espaços.

A Tag PRE serve para pre-formatar o texto do jeito que está no VSCode, usando o mesmo espaçamento, recuos e afastamentos.

O comando SHIFT+TAB faz com que o texto selecionado (podendo ser mais de uma linha) recue.

Citações

A Tag Q adiciona o sinal de ASPAS para a frase dando a semântica de citação. Por exemplo: Meu pai sempre diz: Deus ajuda quem cedo madruga!

Citações Completas

Quando se quer fazer uma citação maior, em que se tem uma quebra de linha e um afastamento para a direita, dando ênfase a citação, utiliza-se a Tag BLOCKQUOTE.

Pode-se usar o parâmetro CITE dentro da tag, e no valor, usar um link para a fonte da citação, não altera o texto, mas ajuda com sites de busca.

Abreviações

Quando em um texto é utilizado alguma abreviação, utiliza-se a TAG ABBR para dar um popup à sigla, escrevendo o significado no parâmetro que aparece após a utilização. Por Exemplo: Vamos ao aeroporto de GYN.

Texto Invertido

Para escrever um texto de forma invertida, utiliza-se as TAGs LTR e RTL. Por Exemplo: Socorram-me, subi no ônibus em Marrocos. Socorram-me, subi no ônibus em Marrocos.
